Midtown Hotel Apps is a mobile hotel booking application designed to make it easy for both local residents and tourists to find and reserve hotel rooms in strategic city locations. The app covers multiple cities in Indonesia — Surabaya, Jakarta, Samarinda, Balikpapan, and Sampit — offering a seamless booking experience without the need to visit the hotel in person.

Built with Flutter, the app provides an intuitive search and reservation flow, room availability browsing, and booking confirmation — all optimized for mobile users. It helps travelers find comfortable, well-located accommodations quickly, making travel planning more convenient and efficient.
