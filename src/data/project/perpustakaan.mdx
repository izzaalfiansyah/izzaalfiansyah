Perpustakaan is a digital library management application developed for SDN Inpres Bhayangkara in Kota Jayapura. The system provides an easy-to-use platform for managing the school's book collection, recording borrowing transactions, and tracking the return status of each book.

The application helps librarians and school staff efficiently manage library resources while enabling students to search for available books and check their borrowing records. By moving library operations online, it promotes better resource utilization and encourages a reading culture within the school community.
