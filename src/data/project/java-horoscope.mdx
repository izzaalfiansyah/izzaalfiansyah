Java Horoscope is a web application built by scraping and reimplementing horoscope and primbon (Javanese fortune-telling) data from primbon.com. The project uses Express.js as the backend API server paired with Puppeteer for automated web scraping, and Vue.js for the frontend interface.

The application allows users to explore Javanese astrological readings and primbon predictions through a clean, modern UI — making traditional cultural knowledge more accessible in a digital format. It serves as both a practical tool and a demonstration of web scraping, API design, and full-stack JavaScript development.
