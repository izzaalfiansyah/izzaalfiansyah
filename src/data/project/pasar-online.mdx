Pasar Online is a web-based online marketplace developed as a final project for a Web Programming course. The application enables buyers and sellers to interact on a digital platform, with features for product listings, browsing, and basic transaction management.

Built with core PHP, the project demonstrates fundamental web development concepts including database integration, session management, and dynamic page rendering — serving as a solid academic foundation for understanding e-commerce systems from the ground up.
